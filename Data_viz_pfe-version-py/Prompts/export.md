# Prompt pour le développement de l'Export Agent avec GPT-4.1
## Focus sur l'export vers RDL (SQL Server Reporting Services) et TWBX (Tableau Packaged Workbook)

---

## Prompt Principal

```
Je souhaite développer un **Export Agent** dans le cadre d'une architecture agentique pour la transformation et génération de visualisations BI. Cet agent est responsable de la génération des artefacts finaux à partir du modèle spécifique à l'outil produit par le Transformation Agent. Je me concentre spécifiquement sur l'export vers deux formats cibles : **RDL** (Report Definition Language pour SQL Server Reporting Services) et **TWBX** (Tableau Packaged Workbook).

## Contexte technique

### Architecture générale
Le système est structuré en trois couches :
- **Interaction Layer** : Conversation Agent
- **Cognition Layer** : Intent Detection Agent + Orchestrator Agent
- **Execution Layer** : Data Extraction, Parsing, Semantic Reasoning, Specification Agent, Transformation Agent, **Export Agent**

### Position de l'Export Agent dans le pipeline
```
Tool-Specific Model (input) → Export Agent → Final Artifact (.rdl, .twbx, .pbix, etc.)
```

L'Export Agent reçoit :
1. **Tool-Specific Model** : modèle intermédiaire produit par le Transformation Agent (structure JSON normalisée)
2. **Target Format** : format de sortie (RDL, TWBX, PBIX, etc.)
3. **Data Sources** : connexions aux données (extract ou live)
4. **Context & Metadata** : métadonnées de transformation, lineage, préférences utilisateur

### Objectifs de l'Export Agent

1. **Génération RDL** : produire des rapports SSRS valides avec paramètres, datasets, et mise en page
2. **Génération TWBX** : créer des packages Tableau complets incluant extracts, métadonnées et visualisations
3. **Optimisation des artefacts** : structurer les fichiers pour performances et maintenabilité
4. **Gestion des dépendances** : intégrer les ressources externes (images, polices, data sources)
5. **Validation de sortie** : vérifier la conformité des fichiers générés
6. **Traçabilité** : embarquer le lineage et les métadonnées dans l'artefact final

class RDLGenerator:
    """Générateur de rapports RDL pour SQL Server Reporting Services"""
    
    def __init__(self, config: RDLConfig):
        self.config = config
        self.xml_builder = XMLBuilder()
        self.schema_version = config.schema_version or "2016/01"
    
    def generate(self, model: ToolSpecificModel) -> str:
        """Génère le contenu XML RDL à partir du modèle"""
        report = self._create_report_element()
        
        # Ajouter les métadonnées
        self._add_metadata(report, model)
        
        # Ajouter les paramètres
        self._add_parameters(report, model.parameters)
        
        # Ajouter les sources de données
        self._add_data_sources(report, model.data_sources)
        
        # Ajouter les datasets
        self._add_datasets(report, model.datasets)
        
        # Ajouter la mise en page
        self._add_layout(report, model.visualizations)
        
        return self.xml_builder.tostring(report, pretty=True)
    
    def _add_layout(self, parent, visualizations: List[Visualization]):
        """Convertit les visualisations en éléments RDL"""
        body = self._create_body()
        
        for viz in visualizations:
            if viz.type == "TABLE":
                self._add_table(body, viz)
            elif viz.type == "CHART":
                self._add_chart(body, viz)
            elif viz.type == "MATRIX":
                self._add_matrix(body, viz)
            elif viz.type == "TEXTBOX":
                self._add_textbox(body, viz)
            elif viz.type == "IMAGE":
                self._add_image(body, viz)
        
        parent.append(body)
    
    def _add_chart(self, parent, viz: Visualization):
        """Ajoute un graphique Chart RDL"""
        chart = ET.Element("Chart", Name=viz.id)
        
        # Configuration du type de graphique
        chart_data = ET.SubElement(chart, "ChartData")
        series_collection = ET.SubElement(chart_data, "ChartSeriesCollection")
        series = ET.SubElement(series_collection, "ChartSeries", Name="Series1")
        
        # Mapping du type de graphique
        subtype_map = {
            "BAR_CHART": "Column",
            "LINE_CHART": "Line",
            "PIE_CHART": "Pie",
            "AREA_CHART": "Area",
            "SCATTER_PLOT": "Scatter"
        }
        ET.SubElement(series, "Subtype").text = subtype_map.get(viz.type, "Column")
        
        # Ajout des points de données
        data_points = ET.SubElement(series, "ChartDataPoints")
        for data_point in viz.data_points:
            point = ET.SubElement(data_points, "ChartDataPoint")
            data_values = ET.SubElement(point, "DataValues")
            
            ET.SubElement(data_values, "DataValue", Name="X").text = f"=Fields!{data_point.x}.Value"
            ET.SubElement(data_values, "DataValue", Name="Y").text = f"=Fields!{data_point.y}.Value"
        
        # Axes
        if viz.x_axis:
            category_axis = ET.SubElement(chart, "ChartCategoryAxis")
            axis = ET.SubElement(category_axis, "Axis")
            ET.SubElement(axis, "Title").text = viz.x_axis.label
        
        if viz.y_axis:
            value_axis = ET.SubElement(chart, "ChartValueAxis")
            axis = ET.SubElement(value_axis, "Axis")
            ET.SubElement(axis, "Title").text = viz.y_axis.label
        
        parent.append(chart)
```

### Gestion des paramètres RDL

```python
class RDLParameterHandler:
    """Gestionnaire des paramètres SSRS"""
    
    def generate_parameters(self, parameters: List[Parameter]) -> List[ET.Element]:
        """Génère les éléments ReportParameter"""
        param_elements = []
        
        for param in parameters:
            param_elem = ET.Element("ReportParameter", Name=param.name)
            
            # Type de données
            data_type = ET.SubElement(param_elem, "DataType")
            data_type.text = self._map_data_type(param.data_type)
            
            # Valeur par défaut
            if param.default_value:
                default_values = ET.SubElement(param_elem, "DefaultValues")
                default_value = ET.SubElement(default_values, "Value")
                default_value.text = str(param.default_value)
            
            # Valeurs disponibles (si dropdown)
            if param.available_values:
                valid_values = ET.SubElement(param_elem, "ValidValues")
                dataset_ref = ET.SubElement(valid_values, "DataSetReference")
                ET.SubElement(dataset_ref, "DataSetName").text = param.value_dataset
                ET.SubElement(dataset_ref, "ValueField").text = param.value_field
                ET.SubElement(dataset_ref, "LabelField").text = param.label_field
            
            # Nullable
            ET.SubElement(param_elem, "Nullable").text = str(param.nullable).lower()
            
            # Prompt
            if param.prompt:
                ET.SubElement(param_elem, "Prompt").text = param.prompt
            
            param_elements.append(param_elem)
        
        return param_elements
    
    def _map_data_type(self, data_type: str) -> str:
        """Map le type abstrait vers le type SSRS"""
        mapping = {
            "STRING": "String",
            "INTEGER": "Integer",
            "DECIMAL": "Float",
            "DATE": "DateTime",
            "DATETIME": "DateTime",
            "BOOLEAN": "Boolean"
        }
        return mapping.get(data_type, "String")
```

## Partie 2 : Export vers TWBX (Tableau Packaged Workbook)

### Structure TWBX

TWBX est un fichier ZIP contenant :
- `twb` : fichier XML Tableau Workbook
- `Data` : dossier contenant les extracts (`.hyper` ou `.tde`)
- `Images` : ressources images
- `Preferences` : fichiers de préférences
- `Metadata` : métadonnées

### Implémentation du générateur TWBX

```python
import zipfile
import os
import tempfile
from pathlib import Path
from typing import Dict, List, Optional
import xml.etree.ElementTree as ET

class TWBXGenerator:
    """Générateur de packages Tableau (TWBX)"""
    
    def __init__(self, config: TWBXConfig):
        self.config = config
        self.temp_dir = tempfile.mkdtemp()
        self.workbook_path = Path(self.temp_dir) / "workbook.twb"
        self.data_dir = Path(self.temp_dir) / "Data"
        self.images_dir = Path(self.temp_dir) / "Images"
        
    def generate(self, model: ToolSpecificModel, extract_data: Optional[pd.DataFrame] = None) -> bytes:
        """Génère le fichier TWBX complet"""
        
        # Créer la structure de dossiers
        self.data_dir.mkdir(exist_ok=True)
        self.images_dir.mkdir(exist_ok=True)
        
        # 1. Générer le fichier TWB
        twb_content = self._generate_twb(model)
        with open(self.workbook_path, 'w', encoding='utf-8') as f:
            f.write(twb_content)
        
        # 2. Générer les extracts Hyper si nécessaire
        if extract_data is not None and self.config.include_extract:
            self._generate_hyper_extract(model, extract_data)
        
        # 3. Copier les ressources
        self._copy_resources(model)
        
        # 4. Créer l'archive ZIP
        return self._create_twbx_archive()
    
    def _generate_twb(self, model: ToolSpecificModel) -> str:
        """Génère le fichier TWB XML"""
        workbook = ET.Element("workbook", 
                              version="18.1",
                              source_build="2023.3",
                              source_platform="win")
        
        # Ajouter les métadonnées
        self._add_metadata(workbook, model)
        
        # Ajouter les datasources
        self._add_datasources(workbook, model.datasources)
        
        # Ajouter les worksheets
        self._add_worksheets(workbook, model.visualizations)
        
        # Ajouter les dashboards
        self._add_dashboards(workbook, model.dashboards)
        
        # Formater le XML
        return self._pretty_xml(workbook)
    
    def _add_datasources(self, parent, datasources: List[DataSource]):
        """Ajoute les sources de données Tableau"""
        datasources_elem = ET.SubElement(parent, "datasources")
        
        for ds in datasources:
            ds_elem = ET.SubElement(datasources_elem, "datasource",
                                     caption=ds.caption,
                                     inline="true",
                                     name=ds.name)
            
            # Configuration de la connexion
            conn = ET.SubElement(ds_elem, "connection",
                                class_="hyper" if ds.use_extract else "sqlserver",
                                dbname=ds.database,
                                server=ds.server)
            
            # Relations
            relation = ET.SubElement(conn, "relation",
                                    connection=ds.name,
                                    name=ds.table_name,
                                    table=f"[{ds.extract_name}]" if ds.use_extract else ds.table_name)
            
            # Métadonnées des colonnes
            metadata = ET.SubElement(relation, "metadata-records")
            for column in ds.columns:
                self._add_column_metadata(metadata, column)
    
    def _add_column_metadata(self, parent, column: Column):
        """Ajoute les métadonnées d'une colonne Tableau"""
        record = ET.SubElement(parent, "metadata-record", class_="column")
        attributes = ET.SubElement(record, "attributes")
        
        # Attributs obligatoires Tableau
        ET.SubElement(attributes, "attribute", name="local-name").text = column.name
        ET.SubElement(attributes, "attribute", name="name").text = f"[{column.name}]"
        ET.SubElement(attributes, "attribute", name="remote-name").text = column.remote_name or column.name
        
        # Type de données
        datatype = self._map_data_type(column.data_type)
        ET.SubElement(attributes, "attribute", name="datatype").text = datatype
        
        # Rôle (dimension/measure)
        role = "dimension" if column.is_dimension else "measure"
        ET.SubElement(attributes, "attribute", name="role").text = role
        
        # Type (nominal, quantitative, temporal)
        if column.is_dimension:
            field_type = "nominal" if column.data_type == "string" else "temporal" if column.is_date else "nominal"
        else:
            field_type = "quantitative"
        ET.SubElement(attributes, "attribute", name="type").text = field_type
        
        # Agrégation par défaut pour les mesures
        if not column.is_dimension and column.default_aggregation:
            ET.SubElement(attributes, "attribute", name="aggregation").text = column.default_aggregation
    
    def _add_worksheets(self, parent, visualizations: List[Visualization]):
        """Ajoute les feuilles de calcul Tableau"""
        worksheets_elem = ET.SubElement(parent, "worksheets")
        
        for viz in visualizations:
            ws = ET.SubElement(worksheets_elem, "worksheet", name=viz.title or viz.id)
            
            # Layout
            layout = ET.SubElement(ws, "layout")
            size = ET.SubElement(layout, "size")
            ET.SubElement(size, "width").text = str(viz.width or 800)
            ET.SubElement(size, "height").text = str(viz.height or 600)
            
            # Table (visualisation)
            table = ET.SubElement(ws, "table")
            view = ET.SubElement(table, "view")
            
            # Datasource
            datasources = ET.SubElement(view, "datasources")
            ET.SubElement(datasources, "datasource", name=viz.datasource_name)
            
            # Pane
            pane = ET.SubElement(view, "panes")
            pane_elem = ET.SubElement(pane, "pane")
            selection = ET.SubElement(pane_elem, "selection")
            mark = ET.SubElement(selection, "mark")
            
            # Type de marque (basé sur le type de visualisation)
            mark_type = self._map_mark_type(viz.type)
            ET.SubElement(mark, "type").text = mark_type
            
            # Encodages
            encodings = ET.SubElement(selection, "encodings")
            self._add_encodings(encodings, viz)
            
            # Rows et Columns
            rows = ET.SubElement(view, "rows")
            cols = ET.SubElement(view, "cols")
            self._add_row_col_levels(rows, cols, viz)
    
    def _add_encodings(self, parent, viz: Visualization):
        """Ajoute les encodages Tableau (colonnes, rangées, couleurs, etc.)"""
        
        # Encoding pour Colonne (X Axis)
        if viz.x_axis:
            encoding = ET.SubElement(parent, "encoding", name="Column", type="dimension")
            ET.SubElement(encoding, "field").text = f"[{viz.x_axis.field}]"
        
        # Encoding pour Rangée (Y Axis)
        if viz.y_axis:
            encoding = ET.SubElement(parent, "encoding", name="Row", type="measure")
            ET.SubElement(encoding, "field").text = f"[{viz.y_axis.field}]"
            if viz.y_axis.aggregation:
                encoding.set("aggregation", viz.y_axis.aggregation)
        
        # Encoding pour Couleur
        if viz.color_field:
            encoding = ET.SubElement(parent, "encoding", name="Color", type="dimension")
            ET.SubElement(encoding, "field").text = f"[{viz.color_field}]"
        
        # Encoding pour Taille
        if viz.size_field:
            encoding = ET.SubElement(parent, "encoding", name="Size", type="measure")
            ET.SubElement(encoding, "field").text = f"[{viz.size_field}]"
        
        # Encoding pour Détail
        if viz.detail_fields:
            for field in viz.detail_fields:
                encoding = ET.SubElement(parent, "encoding", name="Detail", type="dimension")
                ET.SubElement(encoding, "field").text = f"[{field}]"
    
    def _generate_hyper_extract(self, model: ToolSpecificModel, data: pd.DataFrame):
        """Génère un extract Hyper pour Tableau"""
        # Utilisation de la librairie tableauhyperapi ou pandas_to_hyper
        try:
            from tableauhyperapi import HyperProcess, Connection, Telemetry, CreateMode
            
            extract_path = self.data_dir / f"{model.datasources[0].name}.hyper"
            
            with HyperProcess(Telemetry.SEND_USAGE_DATA_TO_TABLEAU) as hyper:
                with Connection(endpoint=hyper.endpoint,
                               database=extract_path,
                               create_mode=CreateMode.CREATE_AND_REPLACE) as connection:
                    
                    # Créer la table
                    table_def = self._create_table_definition(model)
                    connection.catalog.create_table(table_def)
                    
                    # Insérer les données
                    with Connection(endpoint=hyper.endpoint, database=extract_path) as conn:
                        conn.execute_command("COPY ... FROM ...")  # À implémenter
            
            return extract_path
            
        except ImportError:
            # Fallback: générer un fichier CSV (moins performant mais compatible)

    """Validateur pour les fichiers TWBX"""
    
    def validate(self, content: bytes) -> ValidationResult:
        """Valide la structure TWBX"""
        errors = []
        warnings = []
        
        try:
            import zipfile
            import io
            
            # Vérifier que c'est une archive ZIP valide
            zip_file = zipfile.ZipFile(io.BytesIO(content))
            
            # Vérifier les fichiers obligatoires
            required_files = ["workbook.twb"]
            for required in required_files:
                if required not in zip_file.namelist():
                    errors.append(f"Missing required file: {required}")
            
            # Vérifier le TWB
            if "workbook.twb" in zip_file.namelist():
                twb_content = zip_file.read("workbook.twb")
                twb_validation = self._validate_twb(twb_content)
                errors.extend(twb_validation.errors)
                warnings.extend(twb_validation.warnings)
            
            # Vérifier les extraits Hyper
            hyper_files = [f for f in zip_file.namelist() if f.endswith(".hyper")]
            if hyper_files:
                for hyper_file in hyper_files:
                    # Validation basique de l'extrait
                    if zip_file.getinfo(hyper_file).file_size == 0:
                        warnings.append(f"Empty hyper extract: {hyper_file}")
            
        except zipfile.BadZipFile:
            errors.append("Invalid ZIP archive")
        except Exception as e:
            errors.append(f"Validation error: {e}")
        
        return ValidationResult(
            is_valid=len(errors) == 0,
            errors=errors,
            warnings=warnings
        )
    
    def _validate_twb(self, content: bytes) -> ValidationResult:
        """Valide le contenu TWB XML"""
        errors = []
        warnings = []
        
        try:
            import xml.etree.ElementTree as ET
            root = ET.fromstring(content)
            
            # Vérifier l'élément racine
            if root.tag != "workbook":
                errors.append("Root element must be 'workbook'")
            
            # Vérifier les datasources
            if root.find(".//datasources") is None:
                errors.append("No datasources defined")
            
            # Vérifier les worksheets
            worksheets = root.findall(".//worksheet")
            if not worksheets:
                warnings.append("No worksheets defined")
            
            # Vérifier les références aux datasources
            datasource_names = {ds.get("name") for ds in root.findall(".//datasource")}
            for worksheet in worksheets:
                ds_ref = worksheet.find(".//datasource")
                if ds_ref is not None and ds_ref.get("name") not in datasource_names:
                    errors.append(f"Worksheet references unknown datasource: {ds_ref.get('name')}")
            
        except ET.ParseError as e:
            errors.append(f"XML parsing error: {e}")
        
        return ValidationResult(
            is_valid=len(errors) == 0,
            errors=errors,
